# V003 — Chain Freeze/Unfreeze: Live Spectral Performance

*Recorded: 2026-03-14 | Engine: Obscura | Ghost: Don Buchla*

## The Vision

Obscura's physics-based excitation/resonator system should support a performance technique where the player can freeze the current spectral state, layer new excitations on top, then selectively unfreeze — releasing captured harmonics back into the resonator chain like releasing held water. This creates a live spectral sculpting workflow that is neither sampling nor synthesis, but a third thing.

## Why This Matters

Current freeze effects capture a single moment. Chain freeze/unfreeze treats frozen states as material — stackable layers of captured physics that can be released in sequence, creating temporal structures impossible with either live playing or static freeze. It's a performance technique that could only exist in a physical modeling engine where the "frozen" state is a set of differential equations paused mid-computation.

## The Prophecy

Buchla envisions freeze/unfreeze becoming Obscura's signature performance gesture — the thing no other engine in any software ecosystem can do. It would make Obscura not just a physical modeling synth but a live spectral instrument with its own unique playing technique.
