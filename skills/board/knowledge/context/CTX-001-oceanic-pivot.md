# XOceanic Design Pivot — Swarm to String Ensemble

XOceanic's concept was pivoted from "swarm particle synthesis" (128 boid particles per voice with Craig Reynolds' flocking rules) to "paraphonic string ensemble with bioluminescent chromatophore pedalboard."

## Implications
- Any reference to "swarm", "boid", "flock", "particle", "separation/cohesion/alignment" (as boid params), or "murmuration" in the context of Oceanic is STALE
- The correct identity: divide-down oscillator bank, 6 registration stops, triple-BBD ensemble chorus, 5 serial pedalboard effects (FREEZE/SCATTER/TIDE/ABYSS/MIRROR), chromatophore modulator
- The design spec (`xoceanic_design_spec.md`) is marked SUPERSEDED — do not use it as source of truth
- Actual parameter prefix is `oceanic_` (not `ocean_`)
